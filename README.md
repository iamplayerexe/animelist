> A simple, **offline-first** desktop application built with Electron for managing your personal anime watchlist without the fuss of complex websites. Keep track of seasons, movies, OAVs, specials, and more using manual entry or a faster **automatic mode** powered by a predefined local database. Designed for **Windows, Linux and Mac**.

## ‚ú® Features Checklist

-   [x] üñ•Ô∏è **Cross-Platform:** Native installers for Windows, macOS, and Linux.
-   [x] ‚ú® **Manual Addition:** Add anime quickly (Title, Type, Number, Episodes).
-   [x] üöÄ **Automatic Addition:** Quickly add predefined anime entries (Seasons, Movies, OAVs etc.) from a local database.
-   [x] üé® **Automatic Cover Art:** Fetches cover art from Jikan API for manually added entries.
-   [x] üìä **Status & Progress Tracking:** Set status via dropdowns and track episodes with `+`/`-` buttons.
-   [x] üîç **Filtering:** Sidebar filters for **Status** and **Type**.
-   [x] üåê **Multi-language:** Supports 20 languages via a sidebar dropdown.
-   [x] üíæ **Data Management:** Import/Export (`.json`) & Clear Data options.
-   [x] üîí **Offline First:** Your data is stored locally and securely on your computer.
-   [x] üîÑ **Smart Updates:** Seamless auto-updates for Windows & macOS; update notifications for Linux.

## üìñ How to Use

1.  üñ±Ô∏è **Launch:** Open the app.
2.  ‚ûï **Add Anime:** Click on **`+`** (navbar). You'll be prompted to choose:
    *   **Automatic:**
        1.  Select the main anime title from the grid.
        2.  Select the specific seasons, movies, OAVs, etc., you want to add.
        3.  Click **"Add Selected Entry"**.
    *   **Manual:**
        1.  Fill in the details in the popup.
        2.  Click **"Add"**.
3.  üìù **Manage Cards:**
    *   *Status:* Use the **dropdown** on the card.
    *   *Progress:* Use **`+`**/**`-`** buttons.
    *   *Details:* Click the **"Details"** button to edit episode counts and more.
    *   *Delete:* Click **"Details"** -> **"Delete"** -> Confirm.
4.  üìä **Filter List:**
    *   *Toggle Sidebar:* Click **‚ò∞** button.
    *   *Filter:* Click **Status buttons** or select from the **Type dropdown**.
5.  ‚öôÔ∏è **Data & Settings:**
    *   Click **"Data"** button in the sidebar for Import/Export/Clear options.
    *   Use the **language dropdown** at the bottom of the sidebar.
